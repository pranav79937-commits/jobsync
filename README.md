# jobsync
🚀 JobSync – Smart Job Aggregator

📌 Overview

JobSync is a Streamlit-based web application that helps users find, filter, and track job opportunities in one place.

It combines job search with tracking and analytics to improve the job-hunting experience.

---

🎯 Features

- 🔍 Keyword-based job search
- 🎛️ Filters (location, role)
- 🔽 Sorting (latest / relevance)
- 🧹 Duplicate job removal
- 🔐 User authentication (login/signup)
- ⭐ Save/bookmark jobs
- 🚀 Apply tracking
- 📊 Dashboard with analytics
- 🎯 Skill-match scoring

---

🛠️ Tech Stack

- Python
- Streamlit (UI)
- Pandas (data handling)
- JSON (data storage)

---

⚙️ How It Works

1. User logs in or signs up
2. Enters job keyword
3. System fetches job data
4. Applies filtering and sorting
5. Displays results
6. User can:
   - Save jobs
   - Track applications
7. Dashboard shows user activity
